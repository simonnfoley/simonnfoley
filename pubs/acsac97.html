Simon N. Foley. Supporting Secure Canonical Upgrade Policies in Multilevel Secure Object Stores. 13th Annual Computer Security Applications Conference , San Diego, CA, USA, December 8-12. IEEE Computer Society Press, 1997, pages 69--80.

Abstract Secure canonical upgrade policies are multilevel relabel policies that, under certain conditions, allow high-level subjects to update low-level security labels. This paper describes a scheme whereby these policies can be supported within the Message Filter Model for multilevel secure object-oriented database management systems.
